.. _admin-installation:

Installation
============

To install the extension, perform the following steps:

#. Go to the Extension Manager
#. Install the extension (KF - Contentfilter <kf_categoryfilter>)
#. Load the static template



.. _admin-configuration:

Configuration
-------------

* You can exclude the jquery script in the template-constant-editor



.. _use_the_plugin:

How to use the plugin
---------------------

To use the extension, perform the following steps:

#. Go to a page
#. Add a new content element "Plugin" --> Page/Content Filter
#. Choose the plugin options for a view
#. Use recursive if u need
#. Debug-Mode (shows the <f:debug>{_all}</f:debug> for templating
#. HAVE FUN
