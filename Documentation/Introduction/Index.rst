.. _what-it-does:

What does it do?
----------------

The systemcategorie can be used on every page. This plugin use this information and can build some "hashtags" from
this categories.
The mainoptions:

* List of categories

* List of categories by page

* List of categories from current page

* List of categories as isotope (all Pages need an image or some isotope information)



.. _options:

Options
-------

* Shows all categories from the system

* allow to choose the toplevel and show the child elements (recursive)

* sort by (ID/Name/Pagetree) **ALPHA**
